About ✨ LLMjessica

LLMjessica was created in the spirit of an earlier era of the internet, before Web3, when websites were more personal and creative. I wanted to create a space where I could share my projects, evoking the nostalgia of a different time online.

Seemingly anyone can build a website in minutes with a few clicks with online platforms like Wix or Squarespace. This is great, but I wanted to build something from scratch. Although I am not a front-end developer, I aimed to build a static website that is simple and easy to navigate.

The result is LLMjessica.com, a retro-inspired site, with minimal modern flair. Coded using HTML and CSS, LMMjessica is a static website hosted on GitHub Pages and utilizes GitHub Actions for an automated deployment pipeline.
