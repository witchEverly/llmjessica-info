LLMjessica was created in the spirit of an earlier online era, before Web3, when websites were more personal and creative. I've aimed to design a nostalgic website that is basic but also visually appealing.

Coded using HTML and CSS, the result is LLMjessica.com: a retro-inspired site with minimal modern flair to maintain its nostalgia factor. LLMjessica is a static website hosted on GitHub Pages and uses GitHub Actions for an automated deployment pipeline.
